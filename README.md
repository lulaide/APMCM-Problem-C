# 亚太数模杯 C 题：美国互惠关税的全局影响

本仓库在 `solve/` 下提供 5 个 Jupyter Notebook（problem1–problem5.ipynb），基于 data 目录 2015–2025 年的 HTS 关税文本（管道/逗号分隔自动识别），并生成可复现的情景预测与可视化。建模依赖通过 uv 管理，核心库为 pandas、numpy、matplotlib、seaborn、statsmodels。

## 摘要
- 背景：2025 起美国实施“互惠关税”，平均关税从 2.44% 提升到 20.11%，对逆差国加征 11–50%。
- 目标：评估关税冲击对大豆、汽车、半导体、财政收入及宏观经济的短中期影响，并讨论制造业回流的有效性。
- 方法：关税弹性模型（价格传导+需求弹性）、替代效应分配、简单引力/FDI 迁移逻辑、拉弗曲线式关税收入、宏观传导线性系统；必要处使用合理模拟数据并以 HTS 关税文本校准基线。

## 数据与假设
- 数据：`data/*.txt` 提供 2015–2025 的 HTS 关税；`solve/common.py` 封装加载与关税字段标准化（HTS8 补零、`mfn_ad_val_rate` 数值化）。问题一的进出口数据已全部转换为英文字段与国别名，位于 `问题一中国对其他三国大豆贸易额/data202*.csv`。
- 假设：互惠关税基线 10%，逆差伙伴额外 15–20%；价格传导 60–70%；需求弹性依产业设定（大豆 -1.6、汽车 -1.2、芯片 -0.5~-1.0）；反制税 12%；基线进口额 3.1 万亿美元。

## 问题 1：大豆贸易（完整思路与论文写法）
- 数据与清洗  
  - 来源：`问题一中国对其他三国大豆贸易额/` 目录内 2020–2025 年进出口 CSV（英文列：commodity_code, commodity_name, partner_code, partner_name, usd）。  
  - 取 USA、Brazil、Argentina 三国，金额汇总到 1 亿美元单位，缺失记为 0；HTS 1201 关税文本仅用作背景设定。
  - 处理步骤：读取→按年份与 partner_name 分组求和→透视为 year×country 表，得到进口额矩阵和市场份额。

- 现状分析  
  - 进口额（亿美元，2020→2025）：Brazil 249→314，USA 107→77，Argentina 29→20；2023 后美国份额下滑，巴西抬升。  
  - 市场份额（2025）：Brazil ~72%，USA ~18%，Argentina ~10%。  
  - 可视化插图位置：  
    - 图3：柱状图展示 2020–2025 对三国的进口额对比。  
    - 图4：折线图展示进口额趋势演化。  
    - 图6：堆叠面积展示市场份额演化。

- 模型与关税情景  
  - 设定关税路径：2020–2021 平均 3%，2022–2024 10–12%，2025 互惠关税 25%（基线 10%+附加 15%）。  
  - 价格传导 70%，需求弹性 -1.6；美国大豆需求收缩因子 = 1 + (-1.6) * 0.7 * 0.25 ≈ -0.28。  
  - 替代效应：美国损失的进口额按 60%/40% 由 Brazil/Argentina 吸收，体现产能与物流替代能力。

- 结果与讨论  
  - 关税冲击（以 2025 基线为例）：美国进口额由 ~77 降至 ~31 亿美元；巴西升至 ~362 亿美元，阿根廷升至 ~38 亿美元，总量小幅下降但来源重排。  
  - 市场份额对税率高度敏感，关税抬升驱动供应迅速向南美转移；若进一步加入运输成本与季节性，南美吸收比例可能受限。  
  - 可视化插图位置：  
    - 图5：美国进口额与关税路径的双轴图，用于展示关税变化与进口反应的同步性。  
    - 表1（可在图5后给出）：2025 基线与关税情景下三国进口额对比，呈现绝对值与占比变化。

- 扩展与改进  
  - 若获取到官方月度量价数据，可替换金额为数量和均价，重估价格弹性与替代比例。  
  - 可加入巴西/阿根廷出口上限与物流时滞，对冲高关税导致的极端替代假设。  
  - 结合国内压榨需求与库存数据，可进一步评估国内价格传导与库存调节对进口波动的缓冲作用。

## 问题 2：日本汽车（problem2.ipynb）
- 模型：HTS 8703/8704 基准关税约 2.5%；日本直接出口税升至 28%，日系品牌采用“降价吸收+产地转移”策略，迁移概率由关税差逻辑函数驱动，分流到美国本地与墨西哥（USMCA）。
- 结论：日本直接对美出口量下降约 15%，其中 8–10% 转移到墨西哥、5–7% 转为美国本地生产；美国进口结构中“US_local”占比抬升，墨西哥份额上行，日本品牌在美市场份额总体保持但利润率受压。

## 问题 3：半导体（problem3.ipynb）
- 模型：HTS 8541/8542 基准关税近零；将芯片分高端/中端/成熟三段，假设中国供给依赖度 45/35/25%，对中国高端芯片加征 20% 并叠加出口管制；价格敏感度 -0.5~-1.0，设重回美国制造的拉动系数。
- 结论：高端芯片进口额下降 15–20%，国内份额由 12% 升至约 20%；中端小幅回流；成熟制程几乎不变。国家安全指数（对外依赖度倒数）改善最明显于高端，但总体效率下降（成本上升）。

## 问题 4：关税收入（problem4.ipynb）
- 模型：基线进口 3.1 万亿美元，关税率从 2.44% 升至 20.11%，使用拉弗思路计算短期（弹性 -1.1）与中期（弹性 -1.6）税收；假设 2025–2029 税率缓和至 18%。
- 结论：短期税收在 5.8–6.2 千亿美元区间，显著高于基线；中期因贸易收缩降至 4.0–4.5 千亿美元，存在“超过最优税点后收入下滑”的风险。

## 问题 5：宏观与反制（problem5.ipynb）
- 模型：设置 GDP 增速、CPI、制造业 PMI、失业率、美元指数的基线与冲击（互惠关税 20.11%，反制 12%，价格传导 60%）；线性传导衡量需求拖累、成本推动及制造业回流指数。
- 结论：短期 GDP 增速下修约 0.5 个百分点，CPI 抬升 ~0.4%，PMI 因回流略改善但仍 <50；中期如反制持续，美元走弱、失业率上行，而制造业回流指数上升但不足以抵消总体效率损失。

## 主要启示
- 互惠关税短期提高财政收入、压制进口，但在大豆和汽车领域被替代效应和产地转移迅速对冲。
- 高端半导体最易出现成本上升与贸易萎缩，回流效果有限。
- 宏观层面存在通胀上行与增长放缓的双重压力，制造业回流不足以完全抵消效率损失，政策应与供应链友军合作与有针对性的补贴并举。
